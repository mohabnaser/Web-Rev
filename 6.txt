# Slide 5: Use Strict Directive
- The "use strict" directive indicates that the code should be executed in "strict mode".
- It prevents using undeclared variables.

Code Example:
"use strict";
x = 3.14; // This will cause an error because x is not declared

# Slide 6: Array.forEach()
The forEach() method calls a function (a callback function) once for each array element.

Code Example:
var txt = "";
var numbers = [45, 4, 9, 16, 25];
numbers.forEach(myFunction);

function myFunction(value, index, array) {
  txt = txt + value + "<br>";
}

---

# Slide 7: Array.map()
The map() method creates a new array by performing a function on each array element. It does not change the original array.

Code Example:
var numbers1 = [45, 4, 9, 16, 25];
var numbers2 = numbers1.map(myFunction);

function myFunction(value, index, array) {
  return value * 2;
}

---

# Slide 8: Array.filter()
The filter() method creates a new array with array elements that pass a test.

Code Example:
var numbers = [45, 4, 9, 16, 25];
var over18 = numbers.filter(myFunction);

function myFunction(value, index, array) {
  return value > 18;
}

---

# Slide 9: Array.reduce()
The reduce() method runs a function on each array element to produce (reduce it to) a single value. It works from left-to-right.

Code Example:
var numbers1 = [45, 4, 9, 16, 25];
var sum = numbers1.reduce(myFunction);

function myFunction(total, value, index, array) {
  return total + value;
}

---

# Slide 10: Array.every()
The every() method checks if all array elements pass a test.

Code Example:
var numbers = [45, 4, 9, 16, 25];
var allOver18 = numbers.every(myFunction);

function myFunction(value, index, array) {
  return value > 18;
}

---

# Slide 11: Array.some()
The some() method checks if some array elements pass a test.

Code Example:
var numbers = [45, 4, 9, 16, 25];
var someOver18 = numbers.some(myFunction);

function myFunction(value, index, array) {
  return value > 18;
}

# Slide 12: JSON.parse()
Used to convert data receiving from a web server (string) into a JavaScript object.

Code Example:
var txt = '{"name":"John", "age":30, "city":"New York"}'
var obj = JSON.parse(txt);
document.getElementById("demo").innerHTML = obj.name + ", " + obj.age;

# Slide 13: JSON.stringify()
Used to convert a JavaScript object into a string to send data to a web server.

Code Example:

var myJSON = JSON.stringify(obj);
document.getElementById("demo").innerHTML = myJSON;

# Slide 18-20: Arrow Functions
Arrow functions allow a short syntax for writing function expressions. You don't need the `function` keyword, the `return` keyword, or the curly brackets for single-line statements.

Code Example:
// ES5
var x = function(x, y) { return x * y; }

// ES6
const x = (x, y) => x * y;

# Slide 21: For/of Loop
The for/of statement loops through the values of an iterable object (Arrays, Strings, Maps, NodeLists, etc).

Code Example:
var cars = ['BMW', 'Volvo', 'Mini'];
var x;
for (x of cars) {
  document.write(x + "<br >");
}

# Slide 22-25: JavaScript Classes
ES6 classes are templates for JavaScript Objects. Use the keyword `class` and always add a method named `constructor()`.

Code Example:
class Car {
  constructor(name, year) {
    this.name = name;
    this.year = year;
  }
}
let myCar1 = new Car("Ford", 2014);

# Slide 26-28: JavaScript Promises
A Promise is a JavaScript object that links "Producing Code" and "Consuming Code".
States: Pending, Fulfilled (Resolved), Rejected.

Code Example:
let myPromise = new Promise(function(myResolve, myReject) {
  // "Producing Code" (May take some time)
  myResolve(); // when successful
  myReject();  // when error
});

// "Consuming Code" (Must wait for a fulfilled Promise)
myPromise.then(
  function(value) { /* code if successful */ },
  function(error) { /* code if some error */ }
);

# Slide 29: Default Parameter Values
ES6 allows function parameters to have default values.

Code Example:
function myFunction(x, y = 10) {
  return x + y;
}
myFunction(5); // will return 15

# Slide 30: Function Rest Parameter
The rest parameter (...) allows a function to treat an indefinite number of arguments as an array.

Code Example:
function sum(...args) {
  let sum = 0;
  for (let arg of args) sum += arg;
  return sum;
}
let x = sum(4, 9, 16, 25, 29, 100, 66, 77);

# Slide 71: AJAX Introduction
AJAX = Asynchronous JavaScript And XML.
It is a technique for accessing web servers from a web page asynchronously (without reloading the page).

---

# Slide 72-73: AJAX Example
Code Example:
function loadDoc() {
  var xhttp = new XMLHttpRequest();
  xhttp.onreadystatechange = function() {
    if (this.readyState == 4 && this.status == 200) {
      document.getElementById("demo").innerHTML = this.responseText;
    }
  };
  xhttp.open("GET", "ajax_info.txt", true);
  xhttp.send();
}

---

# Slide 74-75: How AJAX Works (Cycle)
1. An event occurs (button click).
2. XMLHttpRequest object is created.
3. Request sent to server.
4. Server processes request.
5. Server sends response back.
6. JavaScript reads response and updates page content.

[Image Description: A diagram showing the cycle between the Browser (Client) and the Server using an XMLHttpRequest object.]

# Slide 85: The Fetch API
The Fetch API interface allows web browser to make HTTP requests to web servers. No need for XMLHttpRequest anymore.

Code Example:
fetch(file)
  .then(x => x.text())
  .then(y => document.getElementById("demo").innerHTML = y);
